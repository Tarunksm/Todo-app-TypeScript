# Todo App

A simple Todo application built with TypeScript, Tailwind CSS, and Vite.

## Features

- Add and remove todos
- Mark todos as completed
- Save todos using Local Storage
- Add todos with the Enter key

## Tech Stack

- TypeScript
- HTML
- Tailwind CSS
- Vite

## Run Locally

```bash
npm install
npm run dev
```
